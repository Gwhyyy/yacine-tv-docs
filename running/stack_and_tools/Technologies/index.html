Technologies Used.

Flutter​




The actual app and admin panel were built using version 3.3 of the Flutter framework, which is an open-source mobile application development framework created by Google. It uses the Dart programming language and allows for the development of high-performance, high-fidelity apps for Android and iOS. You will need to have the Flutter SDK installed on your development environment in order to run the app.

You can follow the official installation of Flutter guide from here.

tip

To verify that is installed successfully, open a command line and run flutter --version, it should output/show you the version if the installed SDK

Otherwise, please consider re-following its installation documentation steps, you can also contact us to reach you and help you ( only for customers ).


You can also follow with this great tutorial about installing Flutter, Android Studio, and Vscode.

Dart​




Dart is the programming language used in Flutter, which is an object-oriented language with C-style syntax . It is used to build high-performance, high-fidelity apps for web, server, and mobile. You will need to have the Dart SDK installed on your development environment in order to run the app.

You can follow the official installation guide of Dart from here.

Node.js​




Node.js is an open-source, JavaScript runtime environment that executes JavaScript code outside of a browser. It allows the use JavaScript on the server side to build fast and scalable network applications, it's is necessary for building the backend. of the project.

Th project's backend was built with Node.js v18.12.1.

You can install it it by following it's official installation guide from here

Express​




The project backend is built with Express, a minimal and flexible Node.js web application framework, as a backend framework. It allows you to handle HTTP requests and responses, and it is required to handle the API calls.

tip

You can install its dependencies manually and use it globally on your device if you want, but there is more easier way to install it and se it only for running this product, its included in the Backend section of this documentation.

MongoDB​




We are using uses MongoDB, a document-oriented database management system. It allows you to store and retrieve data in a JSON-like format, and it is required to store the data of all aspects in the project, such as channels and events...

You can install its dependencies manually and use it globally on your device if you want, but there is more easier way to install it and se it only for running this product, its included in the Backend section of this documentation.
